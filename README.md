# ISTE 612 - Term Project

Term project repository for ISTE 612.

Providers are in this form:

```yaml
facility: <str>
info:
  phone: [<XXX-XXX-XXXX>]
  fax: [<XXX-XXX-XXXX>]
  website: <subdomain.domain.com>
  person: <entity>
keywords: [<str>,...]
address:
  street:
    line_1: <str>
    line_2: <str>
  city: <city>
  county: <county>
  state: <state>
  zipcode: <zipcode>
category: 
  disability: [<str>,...]
  service: [<str>,...]
misc: 
  content: [<str>,...]
  <key>: <value>
```
